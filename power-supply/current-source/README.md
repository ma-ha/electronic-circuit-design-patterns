# Current Sources

## LED Series Resistor

An LED needs a current source. 
The voltage is not so relevant, as long as it's bigger than the diode voltage drop.

The easiest way is a resistor to limit the current.

![Series Resistor](series-resistor.png)


$` R = ( U - U_{LED} ) / I_{LED} `$
