# Circuit Board Design Patterns

This is a collection of basic building blocks for electronic circuits.

- Connectors
  - Case (front/back)
  - Bus board
  - Flat cable
- Differential Pair
- Double sided PCBs
- Multilayer PCBs
  - Ground plane
  - Power plane
- Raiser 
- Return path
- RF transmission line
- Test point
